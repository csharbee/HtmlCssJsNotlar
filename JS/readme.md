# JavaScript
Js' de value ya type atamış oluyoruz. Variable' a type atamıyoruz. 

#### Falsy Values:
Boolean(0) -> false

Boolean(undefined) -> false

Boolean('') -> false

## Function

#### Declare
```javascript
function Test(variable){
  console.log(variable);
}
Test("hello");
```
#### Expression
```javascript
const Test = function(variable){
  console.log(variable);
}
Test("hello");
```
